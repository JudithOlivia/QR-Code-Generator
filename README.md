# QR Code Generator

QR code generator web app that allows you to create customized QR codes with colors, logos, and emojis.

## 🚀 Features

- **Generate QR codes** from any text or URL
- **Custom colors** - Choose any QR code and background color
- **Add logos** - Upload images to place in the center of your QR code
- **Emoji support** - Use emojis as center icons
- **Multiple sizes** - 256px to 2048px (Small, Medium, Large, Extra Large)
- **Download options** - Save as PNG or JPG format
